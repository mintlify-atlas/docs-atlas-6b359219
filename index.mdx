  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with Jdeflate in minutes</p>
  
  <Steps>
    <Step title="Install the library">
      Build and install Jdeflate using Meson:
      
      ```bash
      meson setup builddir
      meson compile -C builddir
      meson install -C builddir
      ```
      
      Or add it as a Meson subproject to your existing project.
    </Step>
    
    <Step title="Include the headers">
      Add the appropriate headers to your C source file:
      
      ```c
      #include <jdeflate/deflator.h>  // For compression
      #include <jdeflate/inflator.h>  // For decompression
      #include <jdeflate/zstrm.h>     // For high-level API
      ```
    </Step>
    
    <Step title="Compress data">
      Create a deflator instance and compress your data:
      
      ```c
      TDeflator* deflator = deflator_create(0, 6, NULL);
      deflator_setsrc(deflator, source, sourcesize);
      deflator_settgt(deflator, target, targetsize);
      
      if (deflator_deflate(deflator, 1) == DEFLT_OK) {
          // Success! Compressed data is in target buffer
          size_t compressed_size = deflator_tgtend(deflator);
      }
      
      deflator_destroy(deflator);
      ```
    </Step>
  </Steps>
</div>
