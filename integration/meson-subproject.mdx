title: "Meson Subproject"
description: "Use Jdeflate as a Meson subproject with automatic dependency management and fallback support"
---

Meson's subproject system allows you to embed Jdeflate directly into your build without system-wide installation. This is the recommended approach for reproducible builds.

## Quick Start

Add Jdeflate as a Meson dependency with automatic fallback:

```python
deps = []
deps += [dependency('jdeflate', fallback: ['jdeflate', 'jdeflate_dep'])]
```

Meson will first try to find Jdeflate via pkg-config. If not found, it falls back to the subproject.

## Setting Up Subproject Directory

<Steps>
  <Step title="Create Subprojects Directory">
    ```bash
    mkdir -p subprojects
    ```
  </Step>
  
  <Step title="Add Jdeflate as Subproject">
    Choose one of these methods:
    
    <CodeGroup>
    ```bash Git Submodule
    git submodule add https://github.com/Jpn666/jdeflate.git subprojects/jdeflate
    ```
    
    ```bash Clone Directly
    git clone https://github.com/Jpn666/jdeflate.git subprojects/jdeflate
    ```
    
    ```bash Using Meson Wrap (if available)
    meson wrap install jdeflate
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Declare Dependency">
    In your `meson.build`:
    
    ```python
    jdeflate_dep = dependency('jdeflate', 
                              fallback: ['jdeflate', 'jdeflate_dep'])
    ```
  </Step>
</Steps>

## Complete Integration Example

Here's a complete project setup:

```
myproject/
├── meson.build
├── src/
│   └── main.c
└── subprojects/
    └── jdeflate/
        ├── meson.build
        ├── jdeflate/
        └── src/
```

### Root meson.build

```python
project('myapp', 'c',
  version: '1.0.0',
  default_options: ['c_std=c99', 'warning_level=3']
)

# Declare dependency with fallback to subproject
jdeflate_dep = dependency('jdeflate', 
                          fallback: ['jdeflate', 'jdeflate_dep'])

# Build executable
executable('myapp',
  'src/main.c',
  dependencies: [jdeflate_dep],
  install: true
)
```

### src/main.c

```c
#include <jdeflate/deflator.h>
#include <jdeflate/inflator.h>
#include <stdio.h>

int main(void) {
    TDeflator* deflator;
    unsigned char input[] = "Hello from Meson subproject!";
    unsigned char output[256];
    
    deflator = deflator_create(0, 6, NULL);
    if (deflator == NULL) {
        fprintf(stderr, "Failed to create deflator\\n");
        return 1;
    }
    
    deflator_setsrc(deflator, input, sizeof(input) - 1);
    deflator_settgt(deflator, output, sizeof(output));
    
    if (deflator_deflate(deflator, 1) != DEFLT_OK) {
        fprintf(stderr, "Compression failed\\n");
        deflator_destroy(deflator);
        return 1;
    }
    
    printf("Compressed %zu bytes\\n", deflator_tgtend(deflator));
    
    deflator_destroy(deflator);
    return 0;
}
```

## Understanding the Dependency Declaration

From Jdeflate's meson.build (lines 104-115):

```python
common = {
    'include_directories': include_directories('.'), 
    'dependencies': deps
}

if meson.is_subproject():
    if get_option('default_library') == 'static':
        jdeflate_dep = declare_dependency(
            link_with: lib.get_static_lib(), 
            kwargs: common
        )
    endif
    if get_option('default_library') == 'shared':
        jdeflate_dep = declare_dependency(
            link_with: lib.get_shared_lib(), 
            kwargs: common, 
            compile_args: extraargs
        )
    endif
```

This creates the `jdeflate_dep` dependency object that your project uses.

## Controlling Library Type

You can force static or shared linking:

```python
jdeflate_dep = dependency('jdeflate', 
                          fallback: ['jdeflate', 'jdeflate_dep'],
                          default_options: ['default_library=static'])
```

<Note>
This overrides the library type only for the subproject, not your main project.
</Note>

## Advanced Configuration

### Passing Build Options to Subproject

```python
jdeflate_dep = dependency('jdeflate',
                          fallback: ['jdeflate', 'jdeflate_dep'],
                          default_options: [
                              'default_library=static',
                              'buildtype=release',
                              'warning_level=0'
                          ])
```

### Conditional Dependency

```python
jdeflate_required = get_option('compression_support')

jdeflate_dep = dependency('jdeflate',
                          required: jdeflate_required,
                          fallback: ['jdeflate', 'jdeflate_dep'])

if jdeflate_dep.found()
    conf_data.set('HAVE_JDEFLATE', 1)
endif
```

## Multiple Dependencies

Combine Jdeflate with other dependencies:

```python
project('myapp', 'c', default_options: ['c_std=c99'])

deps = []

# Jdeflate for compression
deps += [dependency('jdeflate', fallback: ['jdeflate', 'jdeflate_dep'])]

# Other dependencies
deps += [dependency('threads')]

executable('myapp',
  sources: ['src/main.c', 'src/utils.c'],
  dependencies: deps,
  install: true
)
```

## Cross-Compilation

Meson subprojects work seamlessly with cross-compilation:

```bash
meson setup builddir --cross-file=cross/windows-mingw.txt
meson compile -C builddir
```

Example cross file (cross/windows-mingw.txt):

```ini
[binaries]
c = 'x86_64-w64-mingw32-gcc'
strip = 'x86_64-w64-mingw32-strip'
pkgconfig = 'x86_64-w64-mingw32-pkg-config'

[host_machine]
system = 'windows'
cpu_family = 'x86_64'
cpu = 'x86_64'
endian = 'little'
```

<Note>
The JDEFLATE_DLL macro is automatically defined for Windows targets (see meson.build:99-101).
</Note>

## Version Pinning

To ensure reproducible builds, pin to a specific version:

```bash
cd subprojects/jdeflate
git checkout v1.0.0
cd ../..
git add subprojects/jdeflate
```

Or use a Meson wrap file (subprojects/jdeflate.wrap):

```ini
[wrap-git]
url = https://github.com/Jpn666/jdeflate.git
revision = v1.0.0
depth = 1

[provide]
jdeflate = jdeflate_dep
```

## Verifying Subproject Build

Check which dependencies are being built from source:

```bash
meson introspect builddir --dependencies
```

You should see jdeflate listed with type "subproject".

## Advantages of Subprojects

- **No system installation required**: Perfect for CI/CD and isolated builds
- **Version control**: Pin exact versions for reproducibility
- **Cross-platform**: Works identically on all platforms
- **Automatic dependency resolution**: Meson handles everything
- **Static linking support**: Easy to create fully static binaries

## Common Issues

### Subproject Not Found

If Meson reports "Dependency jdeflate found: NO":

1. Verify subproject directory exists:
   ```bash
   ls subprojects/jdeflate/meson.build
   ```

2. Check fallback declaration matches:
   ```python
   dependency('jdeflate', fallback: ['jdeflate', 'jdeflate_dep'])
   ```

### Wrong Library Type

If the wrong library type is selected:

```python
jdeflate_dep = dependency('jdeflate',
                          fallback: ['jdeflate', 'jdeflate_dep'],
                          default_options: ['default_library=static'])
```

## Example: Complete Application

```c
#include <jdeflate/deflator.h>
#include <jdeflate/inflator.h>
#include <stdio.h>
#include <string.h>

int main(int argc, char** argv) {
    if (argc != 2) {
        fprintf(stderr, "Usage: %s <message>\\n", argv[0]);
        return 1;
    }
    
    const char* input = argv[1];
    size_t input_len = strlen(input);
    
    unsigned char compressed[1024];
    unsigned char decompressed[1024];
    
    // Compress
    TDeflator* deflator = deflator_create(0, 9, NULL);
    deflator_setsrc(deflator, (const unsigned char*)input, input_len);
    deflator_settgt(deflator, compressed, sizeof(compressed));
    
    if (deflator_deflate(deflator, 1) != DEFLT_OK) {
        fprintf(stderr, "Compression failed\\n");
        return 1;
    }
    
    size_t compressed_size = deflator_tgtend(deflator);
    printf("Compressed: %zu -> %zu bytes\\n", input_len, compressed_size);
    deflator_destroy(deflator);
    
    // Decompress
    TInflator* inflator = inflator_create(0, NULL);
    inflator_setsrc(inflator, compressed, compressed_size);
    inflator_settgt(inflator, decompressed, sizeof(decompressed));
    
    if (inflator_inflate(inflator, 1) != INFLT_OK) {
        fprintf(stderr, "Decompression failed\\n");
        return 1;
    }
    
    size_t decompressed_size = inflator_tgtend(inflator);
    printf("Decompressed: %.*s\\n", (int)decompressed_size, decompressed);
    inflator_destroy(inflator);
    
    return 0;
}
```
